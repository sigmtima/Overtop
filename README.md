# Overtop

A 2D top-down shooter built with Unity featuring enemy AI, state machines and upgrade systems.

## Current Features

* Player movement
* Shooting system
* Enemy AI
* Enemy patrol behavior
* Enemy chase behavior
* Enemy attack behavior
* Bullet pooling system
* State Machine architecture for player and enemies

## Technologies

* Unity 6
* C#
* State Machine Pattern
* Object Pooling

## In Development

* Upgrade shop system
* Coin drop system
* UI improvements
* Additional enemy types
* New gameplay mechanics
